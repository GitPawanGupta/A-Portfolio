# A-Portfolio

A modern, professional portfolio website for Archana Sharma - Legal Professional built with React and Tailwind CSS.

## 🌟 Features

- **Fully Responsive** - Works perfectly on mobile, tablet, laptop, and desktop
- **Modern Animations** - Smooth scroll animations, hover effects, and transitions
- **Glass Morphism Design** - Premium glass effect with backdrop blur
- **Interactive Elements** - Expandable sections, skill proficiency bars, and more
- **Floating Shapes** - Animated background elements
- **Scroll Progress Bar** - Visual indicator of page scroll
- **Download Resume** - Print/download functionality
- **Advanced CSS** - Gradient backgrounds, custom animations, and 3D transforms

## 🚀 Technologies Used

- React 18
- Tailwind CSS 3
- PostCSS
- Autoprefixer
- Custom CSS Animations
- Intersection Observer API

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/GitPawanGupta/A-Portfolio.git
cd A-Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```

## 🎨 Design Features

- Premium gradient color schemes
- Custom Google Fonts (Playfair Display, Inter)
- Smooth 60fps animations
- Touch-optimized for mobile devices

## 📱 Responsive Breakpoints

- Mobile: 320px - 640px
- Tablet: 640px - 1024px
- Desktop: 1024px+

You can easily customize the portfolio by editing the component files in the `src/components` directory.
